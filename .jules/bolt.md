## 2024-05-24 - [Avoid Multiple Filter Passes for Stats]
**Learning:** Using multiple `.filter()` calls to count occurrences of different statuses in an array (e.g., in `task-management.tsx`) creates redundant intermediate arrays and iterations (O(kN) time, O(k) memory allocations where k is the number of filters).
**Action:** Replace multiple `.filter().length` calls with a single `for` loop (or `reduce`) that computes all statistics in one O(N) pass with O(1) space, especially inside frequently re-evaluated hooks like `useMemo`.
